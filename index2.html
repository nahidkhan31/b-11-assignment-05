Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element is like ByID, ByClassName, ByTagName, ByQuery Selector, ByQuerySelectorAll. This are the ways to select an element in the DOM.

Question-2: What is the difference between innerHTML, innerTEXT, and textContent?

Usually we input HTML inside the element with innerHTML. To do HTML inside an element requires innerHTML. We use innerTEXT to change the inner value. textContent, this is used to set new elements ignoring all elements even hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where an event listener is attached to a parent element instead of adding multiple listeners to child elements. Where events are child to parent elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a phase of event propagation where an event starts at the target element and moves up the parent elements of the DOM.
